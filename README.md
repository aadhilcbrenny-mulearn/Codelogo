# Coding Club AJCE Logo Generator

This project contains a **Python script that programmatically generates the official logo for the Coding Club of AJCE** using the **Pillow (PIL)** image library.

The script creates a modern circular emblem featuring a coding-themed design with `< />` brackets, institutional branding, and circular typography.

## Logo Concept

The logo represents a **modern and technology-focused identity for the Coding Club of AJCE**.
The circular design symbolizes **unity, collaboration, and continuous learning** within the programming community.

At the center, the `< />` symbol represents **coding and software development**, highlighting the club’s technical focus. The **navy background** conveys professionalism and depth, while **cyan accents** reflect innovation, technology, and digital creativity.

The circular ring containing the text **“Amal Jyothi College of Engineering”** and **“Kanjirapally”** establishes the club’s institutional identity and location. The clean typography and minimal layout emphasize a **modern and professional digital aesthetic**.

---

## Features

* Generates a **high-resolution circular logo**
* Draws **custom arc text along circular paths**
* Includes a **coding bracket symbol `< />`**
* Uses **grid-based background design**
* Creates **PNG outputs with transparent background**
* Fully **script-generated logo (no manual design tools needed)**

---

## Requirements

Python 3.x

Install dependencies:

```bash
pip install pillow
```

---

## How to Run

1. Download or clone the repository.
2. Run the script:

```bash
python logo_generator.py
```

3. The script will generate:

```
coding_club_logo.png
coding_club_logo_transparent.png
```
